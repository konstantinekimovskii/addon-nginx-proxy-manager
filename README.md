# Nginx Proxy Manager — Home Assistant Add-on

[![GitHub Release][releases-shield]][releases]
[![License][license-shield]](LICENSE.md)
[![GitHub Actions][github-actions-shield]][github-actions]

Manage Nginx proxy hosts with a simple, powerful interface.

## About

Fork of [hassio-addons/addon-nginx-proxy-manager](https://github.com/hassio-addons/addon-nginx-proxy-manager).

Maintained independently to keep up with latest NPM releases and HAOS versions.

Supports: `aarch64` (Raspberry Pi 5), `amd64`

## Installation via Custom Repository

1. In Home Assistant: **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
2. Add: `https://github.com/konstantinekimovskii/addon-nginx-proxy-manager`
3. Install **Nginx Proxy Manager**
